Estilo e Tipografia

Estilo e tipografia são os elementos responsáveis por transformar páginas simples em interfaces visuais organizadas, atraentes e fáceis de ler. Com CSS, é possível definir cores, fontes, espaçamentos, bordas e efeitos visuais que melhoram a experiência do usuário e dão identidade ao site. Esses recursos permitem controlar desde pequenos detalhes do texto até a aparência completa dos elementos na página.

Cores

Explicações sobre cores em CSS.

Definir cor do texto

Altera a cor das palavras.

color: white;

Definir cor de fundo

Muda a cor do fundo do elemento.

background-color: blue;

Cor em formato hexadecimal

Define a cor usando valores hexadecimais.

color: #ffffff;

Cor usando valores RGB

Define a cor usando valores RGB.

color: rgb(255, 255, 0);

Cor com transparência (RGBA)

Define a cor com nível de opacidade.

color: rgba(255, 255, 0, 0.5);

Cor em formato HSL

Define a cor usando matiz, saturação e luminosidade.

color: hsl(60, 100%, 50%);

Fundo com gradiente

Cria transição suave entre cores.

background: linear-gradient(to right, black, purple);

Tipografia

Explicações sobre tipografia em CSS.

Definir fonte do texto

Altera a família tipográfica.

font-family: Arial, sans-serif;

Tamanho da fonte

Controla o tamanho do texto.

font-size: 16px;

Texto em negrito

Deixa o texto com maior espessura.

font-weight: bold;

Texto em itálico

Inclina o texto.

font-style: italic;

Transformar letras

Altera a forma como as letras são exibidas.

text-transform: uppercase;

Decoração do texto

Adiciona decorações ao texto.

text-decoration: underline;

Alinhamento do texto

Controla o alinhamento horizontal do texto.

text-align: center;

Espaçamento entre letras

Controla o espaçamento entre caracteres.

letter-spacing: 2px;

Espaçamento entre linhas

Controla o espaçamento vertical do texto.

line-height: 1.5;

Espaçamento

Explicações sobre espaçamento em CSS.

Margem externa

Controla o espaço ao redor do elemento.

margin: 10px;

Margem por lado

Controla a margem individualmente em cada lado.

margin-top: 10px;
margin-right: 10px;
margin-bottom: 10px;
margin-left: 10px;

Espaçamento interno

Controla o espaço dentro do elemento.

padding: 10px;

Padding por lado

Controla o espaçamento interno individualmente.

padding-top: 10px;
padding-right: 10px;
padding-bottom: 10px;
padding-left: 10px;

Bordas

Explicações sobre bordas em CSS.

Largura da borda

Controla a espessura da borda.

border-width: 2px;

Estilo da borda

Define o estilo visual da linha.

border-style: solid;

Cor da borda

Controla a cor da borda.

border-color: red;

Borda completa (atalho)

Define largura, estilo e cor em uma única declaração.

border: 2px solid red;

Cantos arredondados

Define o arredondamento dos cantos.

border-radius: 10px;

Forma circular

Transforma o elemento em círculo.

border-radius: 50%;

Sombras

Explicações sobre sombras em CSS.

Sombra em elementos

Adiciona profundidade ao elemento.

box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3);

Sombra no texto

Adiciona sombra às letras.

text-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3);

Múltiplas sombras

Permite aplicar mais de uma sombra.

box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3),
 -2px -2px 5px rgba(0, 0, 0, 0.3);
