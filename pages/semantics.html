Estrutura e Semântica

O HTML semântico dá significado ao conteúdo da página. Em vez de focar apenas no visual, ele foca no propósito da informação, o que é vital para a acessibilidade e para os motores de busca (SEO).

# A Estrutura Básica (Boilerplate)

Toda página web precisa de uma estrutura inicial. O <head> guarda as configurações ocultas, enquanto o <body> contém o que o usuário realmente vê.

<!-- Declaração do tipo de documento para HTML5 -->
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Minha Primeira Página</title>
</head>
<body>
    <!-- O conteúdo visível vai aqui -->
</body>
</html>

# Construindo o Layout com Semântica

Esqueça a época em que os sites eram feitos apenas com milhares de <div>. Hoje, nós dividimos a página em regiões com significados claros.

<body>
    <!-- Cabeçalho principal do site -->
    <header>
        <h1>Meu Blog Tech</h1>
        <nav>
            <!-- Links de navegação -->
        </nav>
    </header>

    <!-- O conteúdo principal e exclusivo da página -->
    <main>
        <article>
            <h2>Como usar CSS Grid</h2>
            <p>Um guia completo sobre layouts bidimensionais...</p>
        </article>
    </main>

    <!-- Rodapé -->
    <footer>
        <p>© 2023 Meu Blog Tech. Todos os direitos reservados.</p>
    </footer>
</body>

# Textos, Títulos e Ênfase

A hierarquia de títulos (do h1 ao h6) é crucial. Use <strong> para palavras de forte importância (normalmente renderizado em negrito) e <em> para ênfase (renderizado em itálico).

<h2>Linguagens Front-end</h2>
<p>
    O <strong>HTML</strong> constrói a estrutura, o <strong>CSS</strong> cuida do design, e o <em>JavaScript</em> adiciona interatividade.
</p>

# Navegação e Listas

Para criar links que levam a outras páginas, usamos a tag âncora <a>. Para agrupar itens em tópicos, usamos listas.

<!-- Uma lista não ordenada contendo links -->
<ul>
    <li><a href="https://developer.mozilla.org" target="_blank">MDN Web Docs</a></li>
    <li><a href="contato.html">Fale Conosco</a></li>
</ul>
